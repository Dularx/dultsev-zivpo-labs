<p align="justify">Модель Kill Chain, или «Цепочка уничтожения», представляет собой концептуальную схему, которая детально описывает последовательные этапы проведения сложной целевой кибератаки — от первоначальной разведки до достижения злоумышленником конечных целей. Изначально эта модель была заимствована из военной доктрины, где она использовалась для планирования и анализа операций по поражению противника. Однако в контексте информационной безопасности её адаптировала компания Lockheed Martin, создав на её основе знаменитую модель Cyber Kill Chain (CKC). Данная модель стала одним из фундаментальных инструментов в арсенале специалистов по кибербезопасности, поскольку она позволяет не только понять логику действий злоумышленника, но и системно выстроить оборону.</p>
